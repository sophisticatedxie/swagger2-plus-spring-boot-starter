#### 介绍
这个项目一开始的时候主要是迎合前后端分离的狂潮，后端api文档新贵swagger2的诞生和它的UI进阶版knife4j， 为了方便它和springboot项目更进一步的简单集成而萌发的一个想法，目前knife4j的作者也已经推出了knife4j的springboot插件包，感谢开源，感谢swagger。
